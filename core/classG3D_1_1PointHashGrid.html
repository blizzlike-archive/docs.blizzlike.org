Detailed Description

template<class Value, class PosFunc = PositionTrait<Value>, class EqualsFunc = EqualsTrait<Value>, class HashFunc = HashTrait<Vector3int32>>
class G3D::PointHashGrid< Value, PosFunc, EqualsFunc, HashFunc >

Storage of data in a sparse 3D grid of point-based data. The space cost for n elements is O(n). For data with approximately uniform density (with respect to the radius hint), the time cost of searching for neighbors is O(1).

Value must be supported by a G3D::PositionTrait, G3D::EqualsTrait, and G3D::HashFunc. overrides are provided for common G3D classes like G3D::Vector3.

Calls delete on all of the values, which are assumed to be pointers. This is a helper to avoid requiring you to iterate through the data structure, removing and deleting each one. Clears the PointHashGrid at the end.

Using objects (instead of pointers) or reference counted pointers is recommended over using pointers and this deleteAll method.
